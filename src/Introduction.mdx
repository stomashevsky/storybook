<Title>Base UI</Title>
<Subtitle>Design system for building high quality applications</Subtitle>

Base UI is a design system for building high-quality applications, providing styling foundations with Tailwind, CSS variable design tokens, and a library of well-crafted, accessible components. Base UI includes:

- **Design tokens** &ndash; for colors, typography, spacing, sizing, shadows, surfaces, and more.
- **Tailwind 4** &ndash; pre-configured with Base UI's design tokens.
- **Component library** &ndash; high-quality components built on top of Radix for consistent accessibility patterns.
- **Utilities** &ndash; for dark mode, responsiveness layouts, and more.
